# getQualificationInputs

Get a data.frame of qualification inputs with columns 'Project',
'BB-Type', 'BB-Name', and 'Section Reference'

## Usage

``` r
getQualificationInputs(qualificationContent, bbData)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

- bbData:

  A data.frame of Project, BB-Type, BB-Name and Parent-Project

## Value

data.frame with columns 'Project', 'BB-Type', 'BB-Name', and 'Section
Reference'
