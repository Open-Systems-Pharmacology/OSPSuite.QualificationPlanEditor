# getBBFromSnapshot

Get a list of building blocks from a project snapshot

## Usage

``` r
getBBFromSnapshot(snapshotPath, bbType = "Simulations")
```

## Arguments

- snapshotPath:

  A path/URL to a project snapshot (JSON file)

- bbType:

  A building block type (e.g. \`"Simulations"\`, \`"Individuals"\`)

## Value

A list of building block information
