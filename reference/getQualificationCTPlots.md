# getQualificationCTPlots

Extract a data.frame containing comparison time (CT) profile information
from the qualification plan content

## Usage

``` r
getQualificationCTPlots(qualificationContent)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

## Value

data.frame with columns \`Title\`, \`Section Reference\`, \`Simulation
Duration\`, \`TimeUnit\` and plot settings
