# getQualificationGOFPlots

Extract a data.frame containing goodness of fit (GOF) plot information
from the qualification plan content

## Usage

``` r
getQualificationGOFPlots(qualificationContent)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

## Value

data.frame with columns \`Title\`, \`Section Reference\`, \`Artifacts\`,
\`PlotTypes\`, \`Groups\` and plot settings
