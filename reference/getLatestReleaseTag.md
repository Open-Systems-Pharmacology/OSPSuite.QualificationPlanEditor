# getLatestReleaseTag

Get the latest release tag from a GitHub repository

## Usage

``` r
getLatestReleaseTag(owner, repo, includePreReleases)
```

## Arguments

- owner:

  Repository owner

- repo:

  Repository name

- includePreReleases:

  Logical indicating whether to include pre-releases

## Value

Latest release tag name, or NULL if fetch fails
