# getQualificationPKRatio

Extract PK ratio data from a qualification plan and return it as a
data.frame with relevant columns

## Usage

``` r
getQualificationPKRatio(qualificationContent)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

## Value

A data.frame with following columns: \`Title\`, \`Section Reference\`,
\`PK-Parameter\`, \`Artifacts\` and Group settings
