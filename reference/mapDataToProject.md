# mapDataToProject

Map building block or simulation parameter data to a project

## Usage

``` r
mapDataToProject(projectId, data, type)
```

## Arguments

- projectId:

  A character identifier of project

- data:

  A data.frame of Building Block or Simulation Parameter data

- type:

  A character \`"BB"\` for Building Block and \`"SimParam"\` for
  Simulation Parameter data

## Value

A formatted list or data.frame mapped to the Project
