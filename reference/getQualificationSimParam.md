# getQualificationSimParam

Get a data.frame of qualification simulation parameters with columns
'Project', 'Parent Project', 'Parent Simulation', 'Path', and
'TargetSimulation'

## Usage

``` r
getQualificationSimParam(qualificationContent)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

## Value

data.frame with columns 'Project', 'Parent Project', 'Parent
Simulation', 'Path', and 'TargetSimulation'
