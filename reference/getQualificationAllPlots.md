# getQualificationAllPlots

Extract a data.frame containing All Plots information from the
qualification plan content

## Usage

``` r
getQualificationAllPlots(qualificationContent, simulationsOutputs)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

- simulationsOutputs:

  A data.frame of Project, Simulation and Output

## Value

data.frame with columns \`Project\`, \`Simulation\` and \`Section
Reference\`
