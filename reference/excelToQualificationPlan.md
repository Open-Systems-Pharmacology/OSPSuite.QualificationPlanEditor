# excelToQualificationPlan

Export Excel as qualification plan

## Usage

``` r
excelToQualificationPlan(
  excelFile,
  qualificationPlan = "qualification_plan.json"
)
```

## Arguments

- excelFile:

  Path of Excel file defining the qualification plan features

- qualificationPlan:

  Path of qualification plan exported file. Must have \`.json\`
  extension.
