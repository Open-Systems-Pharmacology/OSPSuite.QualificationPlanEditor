# mapToQualification

Map and rename qualification data.frame or list using dictionary

## Usage

``` r
mapToQualification(data, sheetName, qualificationPlanSelector = NA)
```

## Arguments

- data:

  A data.frame or a list

- sheetName:

  Selected Excel sheet name

- qualificationPlanSelector:

  Qualification Plan selector for sub-fields

## Value

A data.frame or a list
