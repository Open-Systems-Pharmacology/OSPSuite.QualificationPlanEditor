# mergeBBData

Merge data.frames of Building Blocks data from snapshots and
qualification

## Usage

``` r
mergeBBData(bbData, qualificationBBData)
```

## Arguments

- bbData:

  A data.frame with columns \`Project\`, \`BB-Type\`, \`BB-Name\`,
  \`Parent-Project\`

- qualificationBBData:

  A data.frame with columns \`Project\`, \`BB-Type\`, \`BB-Name\`,
  \`Parent-Project\`

## Value

A data.frame with columns \`Project\`, \`BB-Type\`, \`BB-Name\`,
\`Parent-Project\`
