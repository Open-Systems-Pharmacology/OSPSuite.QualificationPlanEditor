# getQualificationDDIRatioMapping

Extract a data.frame mapping DDI ratio identifiers to relevant DDI Ratio
fields

## Usage

``` r
getQualificationDDIRatioMapping(qualificationContent)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

## Value

A data.frame with the following columns \`Project\`,
\`Simulation_Control\`, \`Simulation_Treatment\`, \`Output\` and
control/treatment settings
