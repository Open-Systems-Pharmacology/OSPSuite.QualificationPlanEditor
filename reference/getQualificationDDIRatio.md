# getQualificationDDIRatio

Extract DDI ratio data from a qualification plan and return it as a
data.frame with relevant columns

## Usage

``` r
getQualificationDDIRatio(qualificationContent)
```

## Arguments

- qualificationContent:

  Content of a qualification plan

## Value

A data.frame with following columns: \`Title\`, \`Section Ref\`,
\`PK-Parameter\`, \`Plot Type\`, \`Subunits\`, \`Artifacts\` and legend
settings
