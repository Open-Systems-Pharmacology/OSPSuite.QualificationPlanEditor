# excelListingValue

Create an Excel expression for listing values used in dataValidation

## Usage

``` r
excelListingValue(data, columnName, sheetName, additionalRows = 0)
```

## Arguments

- data:

  A data.frame

- columnName:

  Character string. Name of column variable from which to get listing

- sheetName:

  Character string. Name of the sheet from which to get listing

- additionalRows:

  Integer. Additional rows to apply dataValidation

## Value

Character string corresponding to Excel expression
